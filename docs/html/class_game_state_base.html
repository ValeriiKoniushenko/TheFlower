Detailed Description

the base class for working with a game state.

for example you have a main menu and game process. Create two class and inherit from this one to create two game state. Also, you need to register new game state in the enum class CustomEvent - create new state and go to Program.cpp. After that open a function Program::ProcessCustomEvent and register your action like an answer for you Custom Event(e.g. Recreate game state to get new GameState)
